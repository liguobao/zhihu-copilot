# Zhihu Copilot

## 功能

- 自动刷新收件箱未读消息数
- 支持导出回答、文章、想法

## Chrome 安装

- [Chrome Web Store](https://chromewebstore.google.com/detail/pajobpfmbncfooabhbnehdhgalhdollf?utm_source=item-share-cb)
